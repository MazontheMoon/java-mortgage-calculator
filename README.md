# A Java Mortgage Calculator

Built as part of the "Code with Mosh" Java course

## Getting Started

To run the mortgage calculator, follow these steps:

- Create a java project in your PC.
- Go in "src" directory.
- Clone the repository: `https://github.com/MazontheMoon/Mortgage-Calculator`
- Compile the Java program: `javac App.java`
- Run the program: `java App`
